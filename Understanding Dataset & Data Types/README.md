# Understanding Dataset & Data Types

## Aim
To explore the Titanic dataset and understand different data types used in machine learning.

## Dataset
Titanic dataset with 891 passenger records and 12 features.

## Files
- **Titanic-Dataset.csv** – Raw dataset
- **Titanic_Notebook.ipynb** – Dataset exploration using Python
- **Titanic_Dataset_Analysis_Report.pdf** – Detailed analysis report

## Data Types Identified
- Numerical: Age, Fare, SibSp, Parch
- Categorical: Sex, Ticket, Cabin, Embarked, Name
- Binary: Survived
- Ordinal: Pclass

## Key Observations
- Missing values in Age, Cabin, and Embarked
- Mean passenger age ≈ 29.7 years
- Survival rate ≈ 38.4%

## Tools Used
Python, Jupyter Notebook, Pandas

## Conclusion
This task helps in understanding dataset structure and data types before preprocessing and model building.
